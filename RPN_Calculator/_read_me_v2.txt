This program is an implentation of a reverse polish notation calculator, completed in Ocaml

Christopher R Preston, Assignment2 - v2 RPN in Ocaml COP4020 Dr. Dobra Spring 2019

to compile:
ocamlc -o RPNv2 RPN_v2.ml

to run: 
ocamlrun RPNv2

program will pause and wait for user to input an RPN expression;

can also pipe in a text file with a single expression and 
no following whitespace by running ocamlrun RPN < input.txt
